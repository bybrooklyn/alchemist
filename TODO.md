# TODO: PC Transfer & Leptos Setup

The Leptos integration for Alchemist is complete (SSR + Hydration). Follow these steps after transferring the codebase to your PC:

## 1. Install cargo-leptos
Run this on your new machine if you haven't already:
```bash
cargo install cargo-leptos
```

## 2. Start the Development Server
Run this from the project root:
```bash
cargo leptos watch
```
- **Web UI:** [http://127.0.0.1:3000](http://127.0.0.1:3000)
- **Live Reload:** Managed on port 3001.

## 3. Verify the Build
- **Server/SSR:** `cargo check --features ssr`
- **Hydration/WASM:** `cargo check --features hydrate --target wasm32-unknown-unknown`

## 4. Testing the UI
1. Open the browser to [http://127.0.0.1:3000](http://127.0.0.1:3000).
2. Click **Scan Now** to verify server function invocation.
3. Observe the **Recent Jobs** table; it updates in real-time via SSE as jobs change state in the database.
