This guide walks you through the "Setup Wizard" that appears the very first time you open Alchemist. It helps you pick your media folders and tells Alchemist which graphics card to use.

Welcome to Alchemist! This guide will walk you through your first time running Alchemist and getting your media library ready for transcoding.

## The Setup Wizard

When you first access `http://localhost:3000`, you'll be greeted by our setup wizard. This will help you configure the essentials:

### 1. Media Directories
Tell Alchemist where your movies and TV shows are located. These should be the folder paths you set up in Docker (like `/media/movies`).

### 2. Hardware Selection
Alchemist will automatically look for any graphics cards in your computer. You'll be asked to choose your **Preferred Vendor**:
- **NVIDIA (NVENC):** Choose this if you have an NVIDIA card.
- **Intel (QSV):** Choose this for Intel processors with built-in graphics.
- **AMD (VAAPI/AMF):** Choose this if you have an AMD card.
- **Apple (VideoToolbox):** Choose this if you are using a Mac.
- **CPU Fallback:** A safe choice if you don't have a supported graphics card. It's slower but works on every computer.

### 3. Default Transcoding Profile
Choose how you want Alchemist to balance speed and quality:
- **Quality First:** Best looking video, but larger files.
- **Balanced:** The best choice for most people.
- **Space Saver:** Smallest files possible, but takes longer to process.

### 4. Scheduler (Optional)
You can tell Alchemist to only work at certain times (like 11:00 PM to 6:00 AM) so it doesn't slow down your computer while you're using it.

## Finishing Setup

After you click finish, Alchemist will start scanning your folders to see what videos you have. You can watch the progress on the **Dashboard**.
