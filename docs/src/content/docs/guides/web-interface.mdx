The Alchemist web interface is a modern, responsive single-page application built with Astro and React. It provides real-time monitoring and control of the transcoding engine.

## Dashboard

The main command center for Alchemist. At a glance, see:

- **Summary Cards**: Active jobs, completed transcodes, failed jobs, and total processed.
- **Recent Activity Feed**: The last 5 jobs with their status and timestamps.
- **Quick Actions**: Trigger an immediate directory scan or pause/resume the engine.

## Jobs

Comprehensive job management interface:

- **Job Table**: Sort, filter, and search through your transcoding queue.
- **Per-Job Actions**: Cancel active jobs, restart failed ones, or view detailed analysis.
- **Bulk Actions**: Restart all failed jobs or clear the list of completed ones.

## Statistics

Deep dive into transcoding performance:

- **Overview Cards**: Total space saved (GB + percentage), average VMAF score, and more.
- **Daily Activity Chart**: See your transcoding history over the last 30 days.
- **Space Efficiency**: Visual comparison of input vs. output file sizes.
- **Performance Metrics**: Average encoding speed (fps) and compression ratios.

## Library

File browser for your configured media directories:

- Navigate your directory tree and view file metadata (size, codec, bitrate).
- Enqueue individual files for transcoding manually.
- See analysis results without starting an encode.

## Logs

Real-time log viewer powered by Server-Sent Events (SSE):

- **Filter by Type**: INFO, SUCCESS, WARNING, ERROR, and DEBUG.
- **Auto-scroll**: Keeps you updated with the latest events automatically.
- **Search**: Quickly find specific jobs or error messages.

## Settings

Configure the transcoding engine:

- **Output Codec**: Toggle between AV1 and HEVC.
- **Quality Profile**: Choose between Quality, Balanced, and Speed.
- **Concurrent Jobs**: Set how many files to encode at once (1-8).
- **Thresholds**: Adjust minimum reduction and file size requirements.

## Appearance

Customize the UI with over 12 color profiles:

- **Vibrant & Energetic**: Helios Orange, Sunset, Neon, Crimson.
- **Cool & Calm**: Deep Blue, Ocean, Emerald.
- **Soft & Dreamy**: Lavender, Purple.
- **Dark & Minimal**: Midnight (OLED), Monochrome, Dracula.
