The **Library Doctor** is a tool that scans your movies to see if any of them are "broken" or corrupt. It's like a check-up for your hard drive to make sure everything will play perfectly when you sit down to watch.

The Library Doctor performs several checks on each file in your library:
- **Can it play?** Ensures the file isn't empty or totally broken.
- **Is it corrupt?** Looks for "glitches" in the file that might cause it to freeze or skip.
- **Is it supported?** Checks if the video format is one that your TV or phone can actually understand.

## How to use the Doctor

You can start a "check-up" at any time:
1.  Go to the **Dashboard**.
2.  Click the **Run Library Doctor** button.

Alchemist will start looking through your files in the background. You can keep using Alchemist while it works.

## What happens if it finds a "sick" file?

If the Doctor finds a problem, it will let you know:
- **Alerts:** You'll see a warning icon next to the file name.
- **Notifications:** If you set up Discord or phone alerts, you'll get a message.

## Fixing the Problem

Once a broken file is found, you can:
- **Try to Repair:** Alchemist will try to "re-wrap" or transcode the file to fix the glitches.
- **Delete it:** If the file is completely dead, you can remove it to save space.
- **Ignore it:** If you know the file is fine and the Doctor is just being too picky, you can clear the alert.

## Pro Tip
Run the Library Doctor once a month. It's a great way to find out if one of your hard drives is starting to fail before you lose any important files!
