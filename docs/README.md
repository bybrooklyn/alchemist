# Alchemist documentation source

The Markdown under `content/` is the authoritative source for the public
Alchemist manual. Dead Signal Works publishes a generated, tracked mirror at
`https://deadsignal.works/alchemist/docs/`.

Edit documentation here, run `python3 scripts/check_docs.py`, then use the Dead
Signal Works website's `bun run sync:alchemist-docs -- <path-to-alchemist>`
command to refresh the public mirror. Do not edit the generated website copy.
